18  Chapter 18. Multivariable Taylor Approximation and Quadratic Forms

Second-order geometry for optimization, statistics, and machine learning

18.1 Why second-order approximation matters

Linear approximation tells us the best flat approximation to a differentiable function near a point. It answers a first-order question:

If the input changes a little, what is the leading-order change in the output?

Second-order approximation asks a deeper question:

How does the function curve near the point?

This question is central in multivariable calculus. It explains why some critical points are minima, some are maxima, and some are saddles. It also explains the geometry of least-squares loss, maximum likelihood, Newton’s method, local uncertainty in statistical estimation, and the local shape of neural-network loss functions.

The main object of the chapter is the Hessian matrix. The gradient gives a vector of first derivatives. The Hessian gives a matrix of second derivatives. Near a point, the Hessian controls the quadratic part of a function.

NoteLearning goals

By the end of this chapter, you should be able to:

  2. Compute and interpret the Hessian matrix.
  3. Connect quadratic forms \(h^T H h\) with local curvature.
  4. Classify quadratic forms as positive definite, negative definite, indefinite, or semidefinite.
  5. Use eigenvalues of the Hessian to understand bowls, hills, saddles, and flat directions.
  6. Explain why second-order approximation is important in optimization, statistics, and machine learning.

18.2 From linear approximation to quadratic approximation

For a one-variable function, the Taylor approximation at \(x=a\) begins as

\[ f(x) \approx f(a)+f'(a)(x-a)+\frac{1}{2}f''(a)(x-a)^2. \]

The first two terms form the tangent-line approximation. The third term adds curvature.

For a two-variable function \(f(x,y)\) near a point \((a,b)\), the linear approximation is

\[ L(x,y)=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b). \]

The second-order approximation adds all possible second-order terms:

\[ \begin{aligned} T_2(x,y) =&\ f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b) \\ &+\frac{1}{2}f_{xx}(a,b)(x-a)^2 +f_{xy}(a,b)(x-a)(y-b) +\frac{1}{2}f_{yy}(a,b)(y-b)^2. \end{aligned} \]

The mixed term appears with coefficient \(f_{xy}(a,b)\) because the matrix form contains two symmetric mixed terms and the factor \(1/2\) balances them.

TipA compact notation

Let

\[ p=\begin{bmatrix}a\\ b\end{bmatrix}, \qquad x=\begin{bmatrix}x\\ y\end{bmatrix}, \qquad h=x-p=\begin{bmatrix}x-a\\ y-b\end{bmatrix}. \]

Then the quadratic Taylor approximation is

\[ f(p+h)\approx f(p)+\nabla f(p)^T h+\frac{1}{2}h^T H_f(p)h. \]

18.3 The Hessian matrix

For a function \(f(x,y)\) with continuous second partial derivatives, the Hessian matrix is

\[ H_f(x,y)= \begin{bmatrix} f_{xx}(x,y) & f_{xy}(x,y)\\ f_{yx}(x,y) & f_{yy}(x,y) \end{bmatrix}. \]

When the mixed partial derivatives are continuous, Clairaut’s theorem gives \(f_{xy}=f_{yx}\), so the Hessian is symmetric.

For a function \(f(x_1,\ldots,x_n)\), the Hessian is the \(n\times n\) matrix

\[ H_f(x)= \begin{bmatrix} \dfrac{\partial^2 f}{\partial x_1\partial x_1} & \cdots & \dfrac{\partial^2 f}{\partial x_1\partial x_n}\\ \vdots & \ddots & \vdots\\ \dfrac{\partial^2 f}{\partial x_n\partial x_1} & \cdots & \dfrac{\partial^2 f}{\partial x_n\partial x_n} \end{bmatrix}. \]

The Hessian is the derivative of the gradient. The gradient tells which direction increases \(f\) fastest. The Hessian tells how that gradient changes from point to point.

18.4 Worked example: a second-order Taylor polynomial

Consider

\[ f(x,y)=e^{x-y}+xy. \]

Find the second-order Taylor approximation at \((0,0)\).

First compute the value and first derivatives:

\[ f(0,0)=1, \]

\[ f_x=e^{x-y}+y, \qquad f_y=-e^{x-y}+x. \]

So

\[ f_x(0,0)=1, \qquad f_y(0,0)=-1. \]

Now compute the second derivatives:

\[ f_{xx}=e^{x-y}, \qquad f_{xy}=-e^{x-y}+1, \qquad f_{yy}=e^{x-y}. \]

At \((0,0)\),

\[ f_{xx}(0,0)=1, \qquad f_{xy}(0,0)=0, \qquad f_{yy}(0,0)=1. \]

Therefore,

\[ T_2(x,y)=1+x-y+\frac{1}{2}x^2+\frac{1}{2}y^2. \]

The first-order terms create a tilted plane. The quadratic terms bend the approximation upward in both coordinate directions.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-0.8, 0.8, 80)
y = np.linspace(-0.8, 0.8, 80)
X, Y = np.meshgrid(x, y)
F = np.exp(X - Y) + X*Y
L = 1 + X - Y
T2 = 1 + X - Y + 0.5*X**2 + 0.5*Y**2

fig = plt.figure()
ax = fig.add_subplot(projection='3d')
ax.plot_surface(X, Y, F, alpha=0.55, linewidth=0)

18.5 Quadratic forms

A quadratic form in two variables has the form

\[ Q(h_1,h_2)=ah_1^2+2bh_1h_2+ch_2^2. \]

In matrix form,

\[ Q(h)=h^T A h, \qquad h=\begin{bmatrix}h_1\\h_2\end{bmatrix}, \qquad A=\begin{bmatrix}a&b\\b&c\end{bmatrix}. \]

The quadratic term in Taylor’s formula is exactly a quadratic form:

\[ \frac{1}{2}h^T H_f(p)h. \]

Thus, local second-order geometry is linear algebra. The shape of a function near a point is controlled by the matrix \(H_f(p)\).

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 90)
y = np.linspace(-2, 2, 90)
X, Y = np.meshgrid(x, y)
forms = [X**2 + Y**2, -X**2 - Y**2, X**2 - Y**2]
titles = ['positive definite', 'negative definite', 'indefinite']

fig = plt.figure(figsize=(11, 3.5))
for i, (Z, title) in enumerate(zip(forms, titles), start=1):
    ax = fig.add_subplot(1, 3, i, projection='3d')

18.6 Definite, negative definite, indefinite, and semidefinite

A symmetric matrix \(A\) defines a quadratic form \(Q(h)=h^T A h\).

  • \(A\) is positive definite if \(h^T A h>0\) for every nonzero vector \(h\).
  • \(A\) is negative definite if \(h^T A h<0\) for every nonzero vector \(h\).
  • \(A\) is indefinite if \(h^T A h\) is positive in some directions and negative in others.
  • \(A\) is positive semidefinite if \(h^T A h\ge 0\) for every vector \(h\), but may be zero for some nonzero \(h\).
  • \(A\) is negative semidefinite if \(h^T A h\le 0\) for every vector \(h\), but may be zero for some nonzero \(h\).

This classification is geometric:

  • positive definite means bowl-shaped,
  • negative definite means upside-down bowl-shaped,
  • indefinite means saddle-shaped,
  • semidefinite means curved in some directions but flat in others.

18.7 Eigenvalues and principal directions

For a symmetric matrix \(A\), the eigenvectors give the principal directions of the quadratic form, and the eigenvalues give the curvature in those directions.

If \(A\) has orthonormal eigenvectors \(u_1,\ldots,u_n\) with eigenvalues \(\lambda_1,\ldots,\lambda_n\), and if

\[ h=c_1u_1+\cdots+c_nu_n, \]

then

\[ h^T A h=\lambda_1c_1^2+\cdots+\lambda_n c_n^2. \]

This formula is one of the most important bridges between calculus and linear algebra.

ImportantHessian eigenvalues and local shape

At a critical point \(p\), the first-order term disappears because \(\nabla f(p)=0\). Then

\[ f(p+h)-f(p)\approx \frac{1}{2}h^T H_f(p)h. \]

So the eigenvalues of \(H_f(p)\) determine the local shape:

  • all positive eigenvalues: local minimum,
  • all negative eigenvalues: local maximum,
  • both positive and negative eigenvalues: saddle point,
  • at least one zero eigenvalue: higher-order analysis may be needed.
import numpy as np

18.8 The two-variable determinant test revisited

For a two-variable function at a critical point \((a,b)\), the Hessian is

\[ H_f(a,b)= \begin{bmatrix} f_{xx}(a,b)&f_{xy}(a,b)\\ f_{xy}(a,b)&f_{yy}(a,b) \end{bmatrix}. \]

Let

\[ D=f_{xx}(a,b)f_{yy}(a,b)-\bigl(f_{xy}(a,b)\bigr)^2. \]

The determinant test is a convenient two-variable version of the eigenvalue test:

  • If \(D>0\) and \(f_{xx}(a,b)>0\), then \(H_f(a,b)\) is positive definite and \(f\) has a local minimum.
  • If \(D>0\) and \(f_{xx}(a,b)<0\), then \(H_f(a,b)\) is negative definite and \(f\) has a local maximum.
  • If \(D<0\), then \(H_f(a,b)\) is indefinite and \(f\) has a saddle point.
  • If \(D=0\), the test is inconclusive.

The determinant measures whether the two eigenvalues have the same sign or opposite signs. The entry \(f_{xx}\) then tells whether the same sign is positive or negative.

18.9 Worked example: classify critical points by quadratic approximation

Let

\[ f(x,y)=x^3+y^3-3xy+4. \]

First compute the gradient:

\[ f_x=3x^2-3y, \qquad f_y=3y^2-3x. \]

Critical points satisfy

\[ y=x^2, \qquad x=y^2. \]

Substituting gives \(x=x^4\), so \(x=0\) or \(x=1\). Thus the critical points are

\[ (0,0), \qquad (1,1). \]

The Hessian is

\[ H_f(x,y)= \begin{bmatrix} 6x&-3\\ -3&6y \end{bmatrix}. \]

At \((0,0)\),

\[ H_f(0,0)= \begin{bmatrix} 0&-3\\ -3&0 \end{bmatrix}, \qquad D=-9<0. \]

So \((0,0)\) is a saddle point.

At \((1,1)\),

\[ H_f(1,1)= \begin{bmatrix} 6&-3\\ -3&6 \end{bmatrix}, \qquad D=27>0, \qquad f_{xx}(1,1)=6>0. \]

So \((1,1)\) is a local minimum. Its value is

\[ f(1,1)=1+1-3+4=3. \]

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-0.8, 1.8, 120)
y = np.linspace(-0.8, 1.8, 120)

18.10 Taylor approximation in \(\mathbb R^n\)

For \(f:\mathbb R^n\to\mathbb R\), the second-order Taylor approximation near \(p\) is

\[ f(p+h)\approx f(p)+\nabla f(p)^T h+\frac{1}{2}h^T H_f(p)h. \]

This formula is independent of dimension. The only difference is that \(h\) has \(n\) components and \(H_f(p)\) is an \(n\times n\) matrix.

A useful way to read the formula is:

\[ \text{new value} \approx \text{base value} + \text{linear change} + \text{curvature correction}. \]

The first-order term dominates for very small \(h\) unless \(\nabla f(p)=0\). At a critical point, the quadratic term becomes the leading term.

18.11 Example: second-order approximation in three variables

Let

\[ f(x,y,z)=x^2+2y^2+3z^2+xy-yz+2x. \]

At \(p=(0,0,0)\),

\[ f(p)=0, \qquad \nabla f(p)=\begin{bmatrix}2\\0\\0\end{bmatrix}. \]

The Hessian is constant:

\[ H_f= \begin{bmatrix} 2&1&0\\ 1&4&-1\\ 0&-1&6 \end{bmatrix}. \]

For \(h=(h_1,h_2,h_3)^T\),

\[ f(h) = 2h_1+\frac{1}{2}h^T H_fh. \]

In this example, the second-order Taylor approximation is exact because \(f\) is already a quadratic polynomial plus a linear term.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 160)
y = np.linspace(-2, 2, 160)
X, Y = np.meshgrid(x, y)
Z_fixed = 0.4
F = X**2 + 2*Y**2 + 3*Z_fixed**2 + X*Y - Y*Z_fixed + 2*X

fig, ax = plt.subplots()
cs = ax.contour(X, Y, F, levels=24)
ax.clabel(cs, inline=True, fontsize=8)
ax.set_xlabel('x')
ax.set_ylabel('y')
ax.set_title('Slice z = 0.4 of a quadratic function')
ax.set_aspect('equal', adjustable='box')
ax.grid(True, alpha=0.25)

18.12 Least squares as a quadratic form

Second-order geometry is especially clear in least squares.

Suppose \(X\) is a data matrix, \(y\) is a response vector, and \(\beta\) is a parameter vector. The least-squares loss is

\[ L(\beta)=\|y-X\beta\|^2. \]

Expanding,

\[ L(\beta)=y^Ty-2\beta^T X^T y+\beta^T X^T X\beta. \]

This is a quadratic function of \(\beta\). Its gradient is

\[ \nabla L(\beta)=2X^T(X\beta-y), \]

and its Hessian is

\[ H_L(\beta)=2X^T X. \]

The Hessian does not depend on \(\beta\). The eigenvalues of \(2X^T X\) determine the shape of the loss landscape:

  • large eigenvalues mean steep curvature,
  • small eigenvalues mean flat directions,
  • zero eigenvalues mean non-unique least-squares solutions.
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(18)
x_data = np.linspace(-2, 2, 35)
y_data = 1.0 + 1.8*x_data + rng.normal(scale=0.35, size=x_data.shape)

b0 = np.linspace(-0.5, 2.5, 160)
b1 = np.linspace(0.0, 3.6, 160)
B0, B1 = np.meshgrid(b0, b1)
Loss = np.zeros_like(B0)
for xi, yi in zip(x_data, y_data):
    Loss += (yi - (B0 + B1*xi))**2

fig, ax = plt.subplots()
cs = ax.contour(B0, B1, Loss, levels=26)
ax.clabel(cs, inline=True, fontsize=8)
ax.set_xlabel(r'$\beta_0$')
ax.set_ylabel(r'$\beta_1$')
ax.set_title('Least-squares loss contours')
ax.grid(True, alpha=0.25)
plt.show()

Xmat = np.column_stack([np.ones_like(x_data), x_data])
H = 2*Xmat.T @ Xmat
print('Hessian:')
print(H)
print('Hessian eigenvalues:', np.linalg.eigvalsh(H))

18.13 Newton’s method from Taylor approximation

Second-order Taylor approximation also explains Newton’s method for optimization.

Suppose \(f:\mathbb R^n\to\mathbb R\) is smooth and we are at a point \(x_k\). Let \(h\) be a small step. The quadratic model is

\[ m_k(h)=f(x_k)+\nabla f(x_k)^T h+\frac{1}{2}h^T H_f(x_k)h. \]

To minimize this quadratic model, set its gradient with respect to \(h\) equal to zero:

\[ \nabla f(x_k)+H_f(x_k)h=0. \]

Thus the Newton step satisfies

\[ H_f(x_k)h=-\nabla f(x_k), \]

and the next point is

\[ x_{k+1}=x_k+h. \]

When the Hessian is positive definite and the starting point is close enough to a local minimum, Newton’s method can converge very quickly. When the Hessian is indefinite or nearly singular, the Newton step may point toward a saddle or become unstable.

import numpy as np
import matplotlib.pyplot as plt

A = np.array([[6.0, 2.0], [2.0, 2.0]])
b = np.array([-4.0, -1.0])

def fval(x):
    return 0.5*x @ A @ x + b @ x

def grad(x):
    return A @ x + b

x_star = -np.linalg.solve(A, b)
start = np.array([2.5, 2.0])

gd = [start]
alpha = 0.12
for _ in range(18):
    gd.append(gd[-1] - alpha*grad(gd[-1]))
gd = np.array(gd)

newton = [start]
newton.append(start - np.linalg.solve(A, grad(start)))
newton = np.array(newton)

grid = np.linspace(-1.2, 3.0, 200)
X, Y = np.meshgrid(grid, grid)
Z = 0.5*(A[0,0]*X**2 + 2*A[0,1]*X*Y + A[1,1]*Y**2) + b[0]*X + b[1]*Y

fig, ax = plt.subplots()
cs = ax.contour(X, Y, Z, levels=24)
ax.plot(gd[:,0], gd[:,1], marker='o', label='gradient descent')
ax.plot(newton[:,0], newton[:,1], marker='s', label='Newton')
ax.scatter([x_star[0]], [x_star[1]], s=60, label='minimum')
ax.set_aspect('equal', adjustable='box')
ax.set_xlabel('x1')
ax.set_ylabel('x2')
ax.set_title('Optimization paths on a quadratic model')
ax.legend()
ax.grid(True, alpha=0.25)

18.14 Conditioning and narrow valleys

A positive definite Hessian may still create numerical difficulty. If the eigenvalues of \(H\) have very different sizes, then the quadratic bowl is narrow and stretched. The ratio

\[ \kappa(H)=\frac{\lambda_{\max}}{\lambda_{\min}} \]

is the condition number of the Hessian when all eigenvalues are positive.

A large condition number means the function is much steeper in one direction than another. Gradient descent may zigzag in narrow valleys. Newton’s method corrects for curvature by multiplying by \(H^{-1}\), but this can be expensive or unstable in high dimensions.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 220)
y = np.linspace(-3, 3, 220)
X, Y = np.meshgrid(x, y)
Z1 = X**2 + Y**2
Z2 = 25*X**2 + Y**2

fig, axes = plt.subplots(1, 2, figsize=(10, 4))
for ax, Z, title in zip(axes, [Z1, Z2], ['condition number 1', 'condition number 25']):
    cs = ax.contour(X, Y, Z, levels=18)
    ax.clabel(cs, inline=True, fontsize=7)
    ax.set_aspect('equal', adjustable='box')
    ax.set_xlabel('h1')
    ax.set_ylabel('h2')
    ax.set_title(title)
    ax.grid(True, alpha=0.25)
plt.tight_layout()
plt.show()

18.15 Statistical interpretation: likelihood and uncertainty

In statistics, second-order approximation appears in the local behavior of a log-likelihood. Suppose \(\ell(\theta)\) is a log-likelihood and \(\hat\theta\) is a maximum likelihood estimate. Since \(\hat\theta\) is a critical point,

\[ \nabla \ell(\hat\theta)=0. \]

The second-order Taylor approximation is

\[ \ell(\theta)\approx \ell(\hat\theta)+\frac{1}{2}(\theta-\hat\theta)^T H_\ell(\hat\theta)(\theta-\hat\theta). \]

At a local maximum, the Hessian is negative definite. The negative Hessian

\[ - H_\ell(\hat\theta) \]

measures local curvature of the log-likelihood. Sharper curvature means the likelihood drops quickly away from the estimate, indicating smaller uncertainty. Flatter curvature means larger uncertainty.

This idea is one reason quadratic approximation is central in large-sample statistics, Bayesian approximation, and uncertainty quantification.

import numpy as np
import matplotlib.pyplot as plt

mu = np.linspace(-1.5, 2.5, 260)
sigma = np.linspace(0.5, 2.5, 260)
MU, SIGMA = np.meshgrid(mu, sigma)

data = np.array([0.1, 1.0, 0.4, 1.3, 0.7, 0.9])
n = data.size

LogLik = -n*np.log(SIGMA) - np.sum((data[:, None, None] - MU)**2, axis=0)/(2*SIGMA**2)
mu_hat = data.mean()
sigma_hat = np.sqrt(np.mean((data - mu_hat)**2))

fig, ax = plt.subplots()

18.16 Beyond second order

The second-order Taylor approximation is powerful, but it has limitations.

  1. If the Hessian is singular, second-order information may not classify a critical point.
  2. If the point is far from the center of approximation, higher-order terms may dominate.
  3. If the function is not twice continuously differentiable, the Hessian may not describe local geometry reliably.
  4. In high-dimensional models, the Hessian may be too large to store or invert directly.

In modern computation, second-order ideas often appear through approximations: diagonal Hessian approximations, quasi-Newton methods, Gauss-Newton methods, Fisher information, curvature regularization, and low-rank approximations.

18.17 Degenerate examples

The function

\[ f(x,y)=x^4+y^4 \]

has a local minimum at \((0,0)\), but its Hessian at the origin is the zero matrix. The second derivative test gives no information, even though the function is clearly nonnegative.

The function

\[ g(x,y)=x^4-y^4 \]

also has zero Hessian at the origin, but the origin is a saddle point.

Thus, when the Hessian is degenerate, higher-order terms matter.

18.18 Algorithmic summary

To use second-order approximation in practice:

  1. Choose a center point \(p\).
  2. Compute \(f(p)\), \(\nabla f(p)\), and \(H_f(p)\).
  3. Form the local model

\[ T_2(p+h)=f(p)+\nabla f(p)^T h+\frac{1}{2}h^T H_f(p)h. \]

  1. At a critical point, classify the Hessian using eigenvalues or definiteness tests.
  2. Interpret the eigenvectors as principal directions and eigenvalues as curvatures.
  3. Use the approximation only locally unless the function is exactly quadratic.

## Why second-order approximation matters

Linear approximation tells us the best flat approximation to a differentiable function near a point.  It answers a first-order question:

> If the input changes a little, what is the leading-order change in the output?

Second-order approximation asks a deeper question:

> How does the function curve near the point?

This question is central in multivariable calculus.  It explains why some critical points are minima, some are maxima, and some are saddles.  It also explains the geometry of least-squares loss, maximum likelihood, Newton's method, local uncertainty in statistical estimation, and the local shape of neural-network loss functions.

The main object of the chapter is the **Hessian matrix**.  The gradient gives a vector of first derivatives.  The Hessian gives a matrix of second derivatives.  Near a point, the Hessian controls the quadratic part of a function.

## From linear approximation to quadratic approximation

For a one-variable function, the Taylor approximation at $x=a$ begins as

$$
f(x) \approx f(a)+f'(a)(x-a)+\frac{1}{2}f''(a)(x-a)^2.
$$

The first two terms form the tangent-line approximation.  The third term adds curvature.

For a two-variable function $f(x,y)$ near a point $(a,b)$, the linear approximation is

$$
L(x,y)=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b).
$$

The second-order approximation adds all possible second-order terms:

$$
\begin{aligned}
T_2(x,y)
=&\ f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b) \\
&+\frac{1}{2}f_{xx}(a,b)(x-a)^2
+f_{xy}(a,b)(x-a)(y-b)
+\frac{1}{2}f_{yy}(a,b)(y-b)^2.
\end{aligned}
$$

The mixed term appears with coefficient $f_{xy}(a,b)$ because the matrix form contains two symmetric mixed terms and the factor $1/2$ balances them.

::: {.callout-tip}
## A compact notation

Let

$$
p=\begin{bmatrix}a\\ b\end{bmatrix},
\qquad
x=\begin{bmatrix}x\\ y\end{bmatrix},
\qquad
h=x-p=\begin{bmatrix}x-a\\ y-b\end{bmatrix}.
$$

Then the quadratic Taylor approximation is

$$
f(p+h)\approx f(p)+\nabla f(p)^T h+\frac{1}{2}h^T H_f(p)h.
$$
:::

## The Hessian matrix

For a function $f(x,y)$ with continuous second partial derivatives, the **Hessian matrix** is

$$
H_f(x,y)=
\begin{bmatrix}
f_{xx}(x,y) & f_{xy}(x,y)\\
f_{yx}(x,y) & f_{yy}(x,y)
\end{bmatrix}.
$$

When the mixed partial derivatives are continuous, Clairaut's theorem gives $f_{xy}=f_{yx}$, so the Hessian is symmetric.

For a function $f(x_1,\ldots,x_n)$, the Hessian is the $n\times n$ matrix

$$
H_f(x)=
\begin{bmatrix}
\dfrac{\partial^2 f}{\partial x_1\partial x_1} & \cdots & \dfrac{\partial^2 f}{\partial x_1\partial x_n}\\
\vdots & \ddots & \vdots\\
\dfrac{\partial^2 f}{\partial x_n\partial x_1} & \cdots & \dfrac{\partial^2 f}{\partial x_n\partial x_n}
\end{bmatrix}.
$$

The Hessian is the derivative of the gradient.  The gradient tells which direction increases $f$ fastest.  The Hessian tells how that gradient changes from point to point.

## Worked example: a second-order Taylor polynomial

Consider

$$
f(x,y)=e^{x-y}+xy.
$$

Find the second-order Taylor approximation at $(0,0)$.

First compute the value and first derivatives:

$$
f(0,0)=1,
$$

$$
f_x=e^{x-y}+y,
\qquad
f_y=-e^{x-y}+x.
$$

So

$$
f_x(0,0)=1,
\qquad
f_y(0,0)=-1.
$$

Now compute the second derivatives:

$$
f_{xx}=e^{x-y},
\qquad
f_{xy}=-e^{x-y}+1,
\qquad
f_{yy}=e^{x-y}.
$$

At $(0,0)$,

$$
f_{xx}(0,0)=1,
\qquad
f_{xy}(0,0)=0,
\qquad
f_{yy}(0,0)=1.
$$

Therefore,

$$
T_2(x,y)=1+x-y+\frac{1}{2}x^2+\frac{1}{2}y^2.
$$

The first-order terms create a tilted plane.  The quadratic terms bend the approximation upward in both coordinate directions.

## Quadratic forms

A **quadratic form** in two variables has the form

$$
Q(h_1,h_2)=ah_1^2+2bh_1h_2+ch_2^2.
$$

In matrix form,

$$
Q(h)=h^T A h,
\qquad
h=\begin{bmatrix}h_1\\h_2\end{bmatrix},
\qquad
A=\begin{bmatrix}a&b\\b&c\end{bmatrix}.
$$

The quadratic term in Taylor's formula is exactly a quadratic form:

$$
\frac{1}{2}h^T H_f(p)h.
$$

Thus, local second-order geometry is linear algebra.  The shape of a function near a point is controlled by the matrix $H_f(p)$.

## Definite, negative definite, indefinite, and semidefinite

A symmetric matrix $A$ defines a quadratic form $Q(h)=h^T A h$.

- $A$ is **positive definite** if $h^T A h>0$ for every nonzero vector $h$.
- $A$ is **negative definite** if $h^T A h<0$ for every nonzero vector $h$.
- $A$ is **indefinite** if $h^T A h$ is positive in some directions and negative in others.
- $A$ is **positive semidefinite** if $h^T A h\ge 0$ for every vector $h$, but may be zero for some nonzero $h$.
- $A$ is **negative semidefinite** if $h^T A h\le 0$ for every vector $h$, but may be zero for some nonzero $h$.

This classification is geometric:

- positive definite means bowl-shaped,
- negative definite means upside-down bowl-shaped,
- indefinite means saddle-shaped,
- semidefinite means curved in some directions but flat in others.

## Eigenvalues and principal directions

For a symmetric matrix $A$, the eigenvectors give the principal directions of the quadratic form, and the eigenvalues give the curvature in those directions.

If $A$ has orthonormal eigenvectors $u_1,\ldots,u_n$ with eigenvalues $\lambda_1,\ldots,\lambda_n$, and if

$$
h=c_1u_1+\cdots+c_nu_n,
$$

then

$$
h^T A h=\lambda_1c_1^2+\cdots+\lambda_n c_n^2.
$$

This formula is one of the most important bridges between calculus and linear algebra.

::: {.callout-important}
## Hessian eigenvalues and local shape

At a critical point $p$, the first-order term disappears because $\nabla f(p)=0$.  Then

$$
f(p+h)-f(p)\approx \frac{1}{2}h^T H_f(p)h.
$$

So the eigenvalues of $H_f(p)$ determine the local shape:

- all positive eigenvalues: local minimum,
- all negative eigenvalues: local maximum,
- both positive and negative eigenvalues: saddle point,
- at least one zero eigenvalue: higher-order analysis may be needed.
:::

## The two-variable determinant test revisited

For a two-variable function at a critical point $(a,b)$, the Hessian is

$$
H_f(a,b)=
\begin{bmatrix}
f_{xx}(a,b)&f_{xy}(a,b)\\
f_{xy}(a,b)&f_{yy}(a,b)
\end{bmatrix}.
$$

Let

$$
D=f_{xx}(a,b)f_{yy}(a,b)-\bigl(f_{xy}(a,b)\bigr)^2.
$$

The determinant test is a convenient two-variable version of the eigenvalue test:

- If $D>0$ and $f_{xx}(a,b)>0$, then $H_f(a,b)$ is positive definite and $f$ has a local minimum.
- If $D>0$ and $f_{xx}(a,b)<0$, then $H_f(a,b)$ is negative definite and $f$ has a local maximum.
- If $D<0$, then $H_f(a,b)$ is indefinite and $f$ has a saddle point.
- If $D=0$, the test is inconclusive.

The determinant measures whether the two eigenvalues have the same sign or opposite signs.  The entry $f_{xx}$ then tells whether the same sign is positive or negative.

## Worked example: classify critical points by quadratic approximation

Let

$$
f(x,y)=x^3+y^3-3xy+4.
$$

First compute the gradient:

$$
f_x=3x^2-3y,
\qquad
f_y=3y^2-3x.
$$

Critical points satisfy

$$
y=x^2,
\qquad
x=y^2.
$$

Substituting gives $x=x^4$, so $x=0$ or $x=1$.  Thus the critical points are

$$
(0,0),
\qquad
(1,1).
$$

The Hessian is

$$
H_f(x,y)=
\begin{bmatrix}
6x&-3\\
-3&6y
\end{bmatrix}.
$$

At $(0,0)$,

$$
H_f(0,0)=
\begin{bmatrix}
0&-3\\
-3&0
\end{bmatrix},
\qquad
D=-9<0.
$$

So $(0,0)$ is a saddle point.

At $(1,1)$,

$$
H_f(1,1)=
\begin{bmatrix}
6&-3\\
-3&6
\end{bmatrix},
\qquad
D=27>0,
\qquad
f_{xx}(1,1)=6>0.
$$

So $(1,1)$ is a local minimum.  Its value is

$$
f(1,1)=1+1-3+4=3.
$$

## Taylor approximation in $\mathbb R^n$

For $f:\mathbb R^n\to\mathbb R$, the second-order Taylor approximation near $p$ is

$$
f(p+h)\approx f(p)+\nabla f(p)^T h+\frac{1}{2}h^T H_f(p)h.
$$

This formula is independent of dimension.  The only difference is that $h$ has $n$ components and $H_f(p)$ is an $n\times n$ matrix.

A useful way to read the formula is:

$$
\text{new value}
\approx
\text{base value}
+
\text{linear change}
+
\text{curvature correction}.
$$

The first-order term dominates for very small $h$ unless $\nabla f(p)=0$.  At a critical point, the quadratic term becomes the leading term.

## Example: second-order approximation in three variables

Let

$$
f(x,y,z)=x^2+2y^2+3z^2+xy-yz+2x.
$$

At $p=(0,0,0)$,

$$
f(p)=0,
\qquad
\nabla f(p)=\begin{bmatrix}2\\0\\0\end{bmatrix}.
$$

The Hessian is constant:

$$
H_f=
\begin{bmatrix}
2&1&0\\
1&4&-1\\
0&-1&6
\end{bmatrix}.
$$

For $h=(h_1,h_2,h_3)^T$,

$$
f(h) = 2h_1+\frac{1}{2}h^T H_fh.
$$

In this example, the second-order Taylor approximation is exact because $f$ is already a quadratic polynomial plus a linear term.

## Least squares as a quadratic form

Second-order geometry is especially clear in least squares.

Suppose $X$ is a data matrix, $y$ is a response vector, and $\beta$ is a parameter vector.  The least-squares loss is

$$
L(\beta)=\|y-X\beta\|^2.
$$

Expanding,

$$
L(\beta)=y^Ty-2\beta^T X^T y+\beta^T X^T X\beta.
$$

This is a quadratic function of $\beta$.  Its gradient is

$$
\nabla L(\beta)=2X^T(X\beta-y),
$$

and its Hessian is

$$
H_L(\beta)=2X^T X.
$$

The Hessian does not depend on $\beta$.  The eigenvalues of $2X^T X$ determine the shape of the loss landscape:

- large eigenvalues mean steep curvature,
- small eigenvalues mean flat directions,
- zero eigenvalues mean non-unique least-squares solutions.

```{python}
#| label: fig-ch18-least-squares-quadratic-form
#| fig-cap: 'A least-squares loss is a quadratic surface in the model parameters.'
#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(18)
x_data = np.linspace(-2, 2, 35)
y_data = 1.0 + 1.8*x_data + rng.normal(scale=0.35, size=x_data.shape)

b0 = np.linspace(-0.5, 2.5, 160)
b1 = np.linspace(0.0, 3.6, 160)
B0, B1 = np.meshgrid(b0, b1)
Loss = np.zeros_like(B0)
for xi, yi in zip(x_data, y_data):
    Loss += (yi - (B0 + B1*xi))**2

ax.set_ylabel(r'$\beta_1$')
ax.set_title('Least-squares loss contours')
ax.grid(True, alpha=0.25)
plt.show()

Xmat = np.column_stack([np.ones_like(x_data), x_data])
H = 2*Xmat.T @ Xmat

print(H)
print('Hessian eigenvalues:', np.linalg.eigvalsh(H))
```

## Newton's method from Taylor approximation

Second-order Taylor approximation also explains Newton's method for optimization.

Suppose $f:\mathbb R^n\to\mathbb R$ is smooth and we are at a point $x_k$.  Let $h$ be a small step.  The quadratic model is

$$
m_k(h)=f(x_k)+\nabla f(x_k)^T h+\frac{1}{2}h^T H_f(x_k)h.
$$

To minimize this quadratic model, set its gradient with respect to $h$ equal to zero:

$$
\nabla f(x_k)+H_f(x_k)h=0.
$$

Thus the Newton step satisfies

$$
H_f(x_k)h=-\nabla f(x_k),
$$

and the next point is

$$
x_{k+1}=x_k+h.
$$

When the Hessian is positive definite and the starting point is close enough to a local minimum, Newton's method can converge very quickly.  When the Hessian is indefinite or nearly singular, the Newton step may point toward a saddle or become unstable.

```{python}
#| label: fig-ch18-newton-vs-gradient-descent
#| fig-cap: 'Gradient descent and Newton steps on a quadratic loss surface.'
#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

A = np.array([[6.0, 2.0], [2.0, 2.0]])
b = np.array([-4.0, -1.0])

def fval(x):
    return 0.5*x @ A @ x + b @ x

def grad(x):
    return A @ x + b

x_star = -np.linalg.solve(A, b)
start = np.array([2.5, 2.0])

gd = [start]
alpha = 0.12
for _ in range(18):
    gd.append(gd[-1] - alpha*grad(gd[-1]))
gd = np.array(gd)

newton = [start]
newton.append(start - np.linalg.solve(A, grad(start)))
newton = np.array(newton)

grid = np.linspace(-1.2, 3.0, 200)
X, Y = np.meshgrid(grid, grid)
Z = 0.5*(A[0,0]*X**2 + 2*A[0,1]*X*Y + A[1,1]*Y**2) + b[0]*X + b[1]*Y

fig, ax = plt.subplots()
cs = ax.contour(X, Y, Z, levels=24)
ax.plot(gd[:,0], gd[:,1], marker='o', label='gradient descent')
ax.plot(newton[:,0], newton[:,1], marker='s', label='Newton')
ax.scatter([x_star[0]], [x_star[1]], s=60, label='minimum')
ax.set_aspect('equal', adjustable='box')
ax.set_xlabel('x1')
ax.set_ylabel('x2')
ax.set_title('Optimization paths on a quadratic model')
ax.legend()
ax.grid(True, alpha=0.25)
plt.show()
```

## Conditioning and narrow valleys

A positive definite Hessian may still create numerical difficulty.  If the eigenvalues of $H$ have very different sizes, then the quadratic bowl is narrow and stretched.  The ratio

$$
\kappa(H)=\frac{\lambda_{\max}}{\lambda_{\min}}
$$

is the **condition number** of the Hessian when all eigenvalues are positive.

A large condition number means the function is much steeper in one direction than another.  Gradient descent may zigzag in narrow valleys.  Newton's method corrects for curvature by multiplying by $H^{-1}$, but this can be expensive or unstable in high dimensions.

## Statistical interpretation: likelihood and uncertainty

In statistics, second-order approximation appears in the local behavior of a log-likelihood.  Suppose $\ell(\theta)$ is a log-likelihood and $\hat\theta$ is a maximum likelihood estimate.  Since $\hat\theta$ is a critical point,

$$
\nabla \ell(\hat\theta)=0.
$$

The second-order Taylor approximation is

$$
\ell(\theta)\approx \ell(\hat\theta)+\frac{1}{2}(\theta-\hat\theta)^T H_\ell(\hat\theta)(\theta-\hat\theta).
$$

At a local maximum, the Hessian is negative definite.  The negative Hessian

$$
- H_\ell(\hat\theta)
$$

measures local curvature of the log-likelihood.  Sharper curvature means the likelihood drops quickly away from the estimate, indicating smaller uncertainty.  Flatter curvature means larger uncertainty.

This idea is one reason quadratic approximation is central in large-sample statistics, Bayesian approximation, and uncertainty quantification.

```{python}
#| label: fig-ch18-log-likelihood-quadratic-approximation
#| fig-cap: 'Near its maximum, a smooth log-likelihood is often well approximated by a quadratic function.'
#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

mu = np.linspace(-1.5, 2.5, 260)
sigma = np.linspace(0.5, 2.5, 260)
MU, SIGMA = np.meshgrid(mu, sigma)

data = np.array([0.1, 1.0, 0.4, 1.3, 0.7, 0.9])
n = data.size

LogLik = -n*np.log(SIGMA) - np.sum((data[:, None, None] - MU)**2, axis=0)/(2*SIGMA**2)
mu_hat = data.mean()
sigma_hat = np.sqrt(np.mean((data - mu_hat)**2))

fig, ax = plt.subplots()
cs = ax.contour(MU, SIGMA, LogLik, levels=22)
ax.clabel(cs, inline=True, fontsize=7)
ax.scatter([mu_hat], [sigma_hat], s=60)
ax.set_xlabel(r'$\mu$')
ax.set_ylabel(r'$\sigma$')
ax.set_title('Normal log-likelihood contours')
ax.grid(True, alpha=0.25)
plt.show()

print('Estimated maximum near:', (mu_hat, sigma_hat))
```

## Beyond second order

The second-order Taylor approximation is powerful, but it has limitations.

1. If the Hessian is singular, second-order information may not classify a critical point.
2. If the point is far from the center of approximation, higher-order terms may dominate.
3. If the function is not twice continuously differentiable, the Hessian may not describe local geometry reliably.
4. In high-dimensional models, the Hessian may be too large to store or invert directly.

In modern computation, second-order ideas often appear through approximations: diagonal Hessian approximations, quasi-Newton methods, Gauss-Newton methods, Fisher information, curvature regularization, and low-rank approximations.

## Degenerate examples

The function

$$
f(x,y)=x^4+y^4
$$

has a local minimum at $(0,0)$, but its Hessian at the origin is the zero matrix.  The second derivative test gives no information, even though the function is clearly nonnegative.

The function

$$
g(x,y)=x^4-y^4
$$

also has zero Hessian at the origin, but the origin is a saddle point.

Thus, when the Hessian is degenerate, higher-order terms matter.


## Algorithmic summary

To use second-order approximation in practice:

1. Choose a center point $p$.
2. Compute $f(p)$, $\nabla f(p)$, and $H_f(p)$.
3. Form the local model

$$
T_2(p+h)=f(p)+\nabla f(p)^T h+\frac{1}{2}h^T H_f(p)h.
$$

4. At a critical point, classify the Hessian using eigenvalues or definiteness tests.
5. Interpret the eigenvectors as principal directions and eigenvalues as curvatures.
6. Use the approximation only locally unless the function is exactly quadratic.

```{python}
#| label: fig-ch18-numerical-hessian-check
#| fig-cap: 'A finite-difference Hessian can approximate local curvature when symbolic derivatives are unavailable.'
#| fig-width: 6.5
#| fig-height: 4
import numpy as np
import matplotlib.pyplot as plt


def f_num(v):
    x, y = v
    return np.exp(x - y) + x*y


def numerical_hessian(f, p, eps=1e-4):
    p = np.asarray(p, dtype=float)
    n = len(p)
    H = np.zeros((n, n))
    for i in range(n):
        for j in range(n):
            ei = np.zeros(n); ei[i] = 1
            ej = np.zeros(n); ej[j] = 1
            H[i, j] = (
                f(p + eps*ei + eps*ej)
                - f(p + eps*ei - eps*ej)
                - f(p - eps*ei + eps*ej)
                + f(p - eps*ei - eps*ej)
            )/(4*eps**2)
    return H

p = np.array([0.0, 0.0])
H_approx = numerical_hessian(f_num, p)
print(H_approx)

r = np.linspace(1e-5, 1e-2, 50)
errors = []
H_true = np.array([[1.0, 0.0], [0.0, 1.0]])
for eps in r:
    errors.append(np.linalg.norm(numerical_hessian(f_num, p, eps=eps) - H_true))

fig, ax = plt.subplots()
ax.loglog(r, errors, marker='o')
ax.set_xlabel('finite-difference step')
ax.set_ylabel('Hessian approximation error')
ax.set_title('Finite-difference Hessian error')
ax.grid(True, alpha=0.25)
plt.show()
```

## Chapter summary

Second-order Taylor approximation extends tangent-plane approximation by adding curvature.  In several variables, curvature is encoded by the Hessian matrix.  The quadratic term

$$
\frac{1}{2}h^T H_f(p)h
$$

is a quadratic form, so the local geometry of a function is controlled by linear algebra.  At a critical point, the gradient term disappears and the Hessian determines whether the function locally behaves like a bowl, an upside-down bowl, a saddle, or a degenerate flat shape.

This chapter is a bridge from calculus to modern applied mathematics.  Quadratic approximations explain second derivative tests, Newton's method, least-squares geometry, local likelihood approximations, uncertainty quantification, and the curvature of loss landscapes.
