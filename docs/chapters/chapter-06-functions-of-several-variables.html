6  Chapter 6. Functions of Several Variables

Scalar fields, domains, surfaces, level sets, and data landscapes

6.1 Chapter overview

Single-variable calculus studies functions of the form

\[ y=f(x), \]

where one real input produces one real output. Multivariable calculus studies functions such as

\[ z=f(x,y), \qquad w=f(x,y,z), \qquad F=f(x_1,x_2,\ldots,x_n). \]

The input is no longer a point on a line. It may be a point in the plane, a point in space, or a vector in a high-dimensional parameter space. The output is still a scalar: a height, temperature, density, probability, loss, likelihood, energy, cost, or risk.

This chapter builds the language for these objects. We will learn how to describe domains, draw surfaces, read contour maps, study level curves and level surfaces, and use lower-dimensional views to understand high-dimensional functions.

NoteRole in the book

The next several chapters study limits, derivatives, gradients, optimization, and multiple integrals. All of those topics depend on the basic geometric viewpoint developed here: a multivariable function is a scalar field over a region of space.

6.2 Learning goals

After studying this chapter, you should be able to:

  1. define functions of two, three, and \(n\) variables;
  2. distinguish domain, codomain, and range;
  3. find domains defined by square roots, logarithms, denominators, and geometric restrictions;
  4. interpret the graph of \(z=f(x,y)\) as a surface above a domain in the \(xy\)-plane;
  5. recognize planes, hemispheres, paraboloids, and saddle surfaces;
  6. compute and interpret level curves \(f(x,y)=c\);
  7. interpret level surfaces \(f(x,y,z)=c\);
  8. use slices, contour plots, heatmaps, and surface plots to visualize functions;
  9. connect multivariable functions to probability, statistics, optimization, data science, and machine learning.

6.3 6.1 Functions of several variables

A function of two variables is a rule that assigns to each input pair \((x,y)\) in a domain \(D\subseteq \mathbb R^2\) exactly one real number \(f(x,y)\):

\[ f:D\subseteq \mathbb R^2\to \mathbb R. \]

The set \(D\) is the domain. The set of all output values is the range:

\[ \operatorname{Range}(f)=\{f(x,y):(x,y)\in D\}. \]

A function of three variables is a rule

\[ f:E\subseteq \mathbb R^3\to \mathbb R, \qquad (x,y,z)\mapsto f(x,y,z). \]

More generally, a function of \(n\) variables is a rule

\[ f:D\subseteq \mathbb R^n\to \mathbb R, \qquad x=(x_1,x_2,\ldots,x_n)\mapsto f(x_1,x_2,\ldots,x_n). \]

A function \(f:\mathbb R^n\to\mathbb R\) is often called a scalar field because it assigns a scalar value to each point in space.

6.3.1 Examples from science, statistics, and AI

Function Input Output Interpretation
\(T(x,y)\) longitude and latitude temperature weather map
\(p(x,y,z)\) point in space pressure fluid or atmosphere
\(\rho(x,y,z)\) point in space density mass distribution
\(L(\beta_0,\beta_1)\) model parameters squared error loss linear regression
\(\ell(\theta)\) statistical parameter log-likelihood inference
\(J(w_1,\ldots,w_n)\) model weights training objective machine learning
TipModern viewpoint

In data science and AI, the most important functions are often high-dimensional. A neural-network loss function may depend on millions or billions of parameters. We cannot draw its graph, but the ideas of domains, level sets, slices, gradients, and local approximations remain central.

6.4 6.2 Domains as regions

Finding the domain is often the first step in studying a multivariable function. The algebraic restrictions are familiar from single-variable calculus, but now they describe regions in the plane or space.

Common restrictions:

Expression Restriction
\(\sqrt{g(x,y)}\) \(g(x,y)\ge 0\)
\(\ln(g(x,y))\) \(g(x,y)>0\)
\(\dfrac{1}{g(x,y)}\) \(g(x,y)\ne 0\)
\(\arcsin(g(x,y))\) \(-1\le g(x,y)\le 1\)

6.4.1 Worked example 6.1: Square root and denominator

Find the domain of

\[ f(x,y)=\frac{\sqrt{y-x}}{y}. \]

The square root requires

\[ y-x\ge 0, \]

so \(y\ge x\). The denominator requires \(y\ne 0\). Therefore

\[ D=\{(x,y)\in\mathbb R^2:y\ge x,\; y\ne 0\}. \]

This is the half-plane above the line \(y=x\), with the portion of the line \(y=0\) removed.

6.4.2 Worked example 6.2: Parabolic domain with excluded lines

Find the domain of

\[ g(x,y)=\frac{\sqrt{x^2-y}}{x^2-4}. \]

The square root requires

\[ x^2-y\ge 0, \]

so \(y\le x^2\). The denominator requires

\[ x^2-4\ne 0, \]

so \(x\ne 2\) and \(x\ne -2\). Thus

\[ D=\{(x,y)\in\mathbb R^2:y\le x^2,\; x\ne \pm 2\}. \]

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-4, 4, 600)
y = np.linspace(-3, 8, 600)
X, Y = np.meshgrid(x, y)
D = (Y <= X**2)

fig, ax = plt.subplots()
ax.contourf(X, Y, D.astype(float), levels=[-0.5, 0.5, 1.5], alpha=0.55)
ax.plot(x, x**2, linewidth=2, label='$y=x^2$')
ax.axvline(2, linestyle='--', linewidth=1.5, label='$x=2$ removed')
ax.axvline(-2, linestyle='--', linewidth=1.5, label='$x=-2$ removed')
ax.set_xlim(-4, 4)
ax.set_ylim(-3, 8)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('A domain as a region in the plane')
ax.grid(True, alpha=0.25)
ax.legend(loc='upper center', ncols=3, fontsize=8)
plt.show()

6.4.3 Worked example 6.3: Logarithmic disk

Find the domain of

\[ h(x,y)=\ln(9-x^2-y^2). \]

The logarithm requires

\[ 9-x^2-y^2>0. \]

Equivalently,

\[ x^2+y^2<9. \]

Thus the domain is the open disk of radius \(3\) centered at the origin.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3.5, 3.5, 500)
y = np.linspace(-3.5, 3.5, 500)
X, Y = np.meshgrid(x, y)
inside = X**2 + Y**2 < 9

theta = np.linspace(0, 2*np.pi, 400)
fig, ax = plt.subplots()
ax.contourf(X, Y, inside.astype(float), levels=[-0.5, 0.5, 1.5], alpha=0.55)
ax.plot(3*np.cos(theta), 3*np.sin(theta), linestyle='--', linewidth=2)
ax.set_aspect('equal', adjustable='box')
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('Open disk domain')
ax.grid(True, alpha=0.25)
plt.show()
WarningCommon mistake

For \(\ln(9-x^2-y^2)\) the boundary circle \(x^2+y^2=9\) is not included, because \(\ln(0)\) is not defined. For \(\sqrt{9-x^2-y^2}\) the boundary is included, because \(\sqrt{0}=0\).

6.5 6.3 Graphs of functions of two variables

For a function

\[ f:D\subseteq\mathbb R^2\to\mathbb R, \]

the graph is the set of points in \(\mathbb R^3\) given by

\[ \{(x,y,z):(x,y)\in D,\; z=f(x,y)\}. \]

We usually write the graph as

\[ z=f(x,y). \]

The graph is a surface above or below the domain in the \(xy\)-plane.

6.5.1 Worked example 6.4: A plane as a graph

Let

\[ f(x,y)=6-2x-3y. \]

Then

\[ z=6-2x-3y, \]

or equivalently

\[ 2x+3y+z=6. \]

This is a plane. It has \(x\)-intercept \(3\), \(y\)-intercept \(2\), and \(z\)-intercept \(6\).

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-1, 4, 60)
y = np.linspace(-1, 3, 60)
X, Y = np.meshgrid(x, y)
Z = 6 - 2*X - 3*Y

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, alpha=0.75, linewidth=0)
ax.scatter([3, 0, 0], [0, 2, 0], [0, 0, 6], s=40)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_zlabel('$z$')
ax.set_title('Plane $2x+3y+z=6$')
plt.show()

6.5.2 Worked example 6.5: Upper hemisphere

Let

\[ f(x,y)=\sqrt{4-x^2-y^2}. \]

The domain is determined by

\[ 4-x^2-y^2\ge 0, \]

so

\[ x^2+y^2\le 4. \]

The graph satisfies

\[ z=\sqrt{4-x^2-y^2}. \]

Squaring both sides gives

\[ x^2+y^2+z^2=4. \]

Because \(z\ge 0\), the graph is the upper hemisphere of radius \(2\) centered at the origin. The range is

\[ 0\le z\le 2. \]

import numpy as np
import matplotlib.pyplot as plt

r = np.linspace(0, 2, 70)
theta = np.linspace(0, 2*np.pi, 120)
R, T = np.meshgrid(r, theta)
X = R*np.cos(T)
Y = R*np.sin(T)
Z = np.sqrt(np.maximum(0, 4 - X**2 - Y**2))

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, alpha=0.75, linewidth=0)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_zlabel('$z$')
ax.set_title('Upper hemisphere')
plt.show()

6.5.3 Worked example 6.6: Elliptic paraboloid

Let

\[ f(x,y)=x^2+9y^2. \]

The domain is all of \(\mathbb R^2\). Since squares are nonnegative, the range is

\[ z\ge 0. \]

The graph

\[ z=x^2+9y^2 \]

is an elliptic paraboloid. It opens upward, and it rises faster in the \(y\)-direction than in the \(x\)-direction because the coefficient of \(y^2\) is larger.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2.5, 2.5, 90)
y = np.linspace(-1.2, 1.2, 90)
X, Y = np.meshgrid(x, y)
Z = X**2 + 9*Y**2

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, alpha=0.75, linewidth=0)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_zlabel('$z$')
ax.set_title('Elliptic paraboloid')
plt.show()

6.5.4 Worked example 6.7: A saddle surface

Let

\[ f(x,y)=x^2-y^2. \]

The graph

\[ z=x^2-y^2 \]

is a saddle surface. Along the \(x\)-axis, where \(y=0\), it becomes \(z=x^2\), which opens upward. Along the \(y\)-axis, where \(x=0\), it becomes \(z=-y^2\), which opens downward.

Saddle surfaces are essential in optimization because many critical points in high-dimensional loss functions are neither minima nor maxima.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 90)
y = np.linspace(-2, 2, 90)
X, Y = np.meshgrid(x, y)
Z = X**2 - Y**2

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, alpha=0.75, linewidth=0)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_zlabel('$z$')
ax.set_title('Saddle surface')
plt.show()

6.6 6.4 Slices and traces

A surface can be understood by taking lower-dimensional slices.

For \(z=f(x,y)\):

  • fixing \(y=y_0\) gives the curve \(z=f(x,y_0)\) in a vertical plane parallel to the \(xz\)-plane;
  • fixing \(x=x_0\) gives the curve \(z=f(x_0,y)\) in a vertical plane parallel to the \(yz\)-plane;
  • fixing \(z=c\) gives the level curve \(f(x,y)=c\) in the \(xy\)-plane.

Slices connect single-variable calculus to multivariable calculus. Partial derivatives in later chapters measure slopes of coordinate slices.

6.6.1 Worked example 6.8: Slices of an elliptic paraboloid

For

\[ z=x^2+9y^2, \]

fixing \(y=0\) gives

\[ z=x^2, \]

while fixing \(x=0\) gives

\[ z=9y^2. \]

The second slice is steeper.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 300)
y = np.linspace(-1.2, 1.2, 300)

fig, ax = plt.subplots()
ax.plot(x, x**2, label='$y=0$: $z=x^2$')
ax.plot(y, 9*y**2, label='$x=0$: $z=9y^2$')
ax.set_xlabel('free variable')
ax.set_ylabel('$z$')
ax.set_title('Two coordinate slices')
ax.grid(True, alpha=0.25)
ax.legend()
plt.show()

6.7 6.5 Level curves and contour maps

A level curve of a function \(f(x,y)\) is a curve of the form

\[ f(x,y)=c, \]

where \(c\) is a constant in the range of \(f\).

Geometrically, the level curve \(f(x,y)=c\) is obtained by intersecting the surface \(z=f(x,y)\) with the horizontal plane \(z=c\) and projecting the intersection down to the \(xy\)-plane.

Contour maps are collections of level curves. They are used in topography, weather maps, probability density plots, and loss landscapes.

TipHow to read contours

Where level curves are close together, the function changes rapidly. Where level curves are far apart, the function changes slowly.

6.7.1 Worked example 6.9: Circular level curves

Let

\[ f(x,y)=x^2+y^2. \]

The level curve \(f(x,y)=c\) is

\[ x^2+y^2=c. \]

For \(c>0\), this is a circle centered at the origin with radius \(\sqrt c\).

6.7.2 Worked example 6.10: Elliptic level curves

Let

\[ f(x,y)=x^2+9y^2. \]

The level curve \(f(x,y)=c\) is

\[ x^2+9y^2=c. \]

For \(c>0\), these are ellipses.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-4, 4, 400)
y = np.linspace(-2, 2, 400)
X, Y = np.meshgrid(x, y)
Z = X**2 + 9*Y**2

fig, ax = plt.subplots()
cs = ax.contour(X, Y, Z, levels=[1, 4, 9, 16, 25, 36])
ax.clabel(cs, inline=True, fontsize=9)
ax.set_aspect('equal', adjustable='box')
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('Elliptic level curves')
ax.grid(True, alpha=0.25)
plt.show()

6.7.3 Worked example 6.11: Exponential level curves

Let

\[ f(x,y)=xe^{4y}. \]

A level curve \(f(x,y)=c\) satisfies

\[ xe^{4y}=c. \]

If \(x\ne 0\) and \(c/x>0\), then

\[ y=\frac14\ln\left(\frac{c}{x}\right). \]

This shows that level curves need not be lines, circles, parabolas, or ellipses.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 600)
y = np.linspace(-1, 1, 600)
X, Y = np.meshgrid(x, y)
Z = X*np.exp(4*Y)

fig, ax = plt.subplots()
levels = [-4, -2, -1, 1, 2, 4]
cs = ax.contour(X, Y, Z, levels=levels)
ax.clabel(cs, inline=True, fontsize=9)
ax.axvline(0, linewidth=1)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('Exponential level curves')
ax.grid(True, alpha=0.25)
plt.show()

6.8 6.6 Heatmaps and data surfaces

A multivariable function may be known through data rather than a formula. For example, a temperature field may be measured on a grid of locations, or a loss function may be evaluated at many parameter values.

A heatmap colors the plane according to function values. A contour plot draws level curves. Both are useful when a full surface plot is difficult to read.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 300)
y = np.linspace(-3, 3, 300)
X, Y = np.meshgrid(x, y)
Z = np.exp(-0.4*((X-1)**2 + (Y+0.5)**2)) + 0.7*np.exp(-0.8*((X+1.2)**2 + (Y-1.1)**2))

fig, ax = plt.subplots()
image = ax.imshow(Z, extent=[x.min(), x.max(), y.min(), y.max()], origin='lower', aspect='auto')
cs = ax.contour(X, Y, Z, levels=8, linewidths=1)
ax.clabel(cs, inline=True, fontsize=8)
fig.colorbar(image, ax=ax, label='function value')
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('Heatmap with contour lines')
plt.show()

6.9 6.7 Functions of three variables and level surfaces

A function of three variables has the form

\[ f(x,y,z). \]

Its graph would be

\[ w=f(x,y,z), \]

which lives in four-dimensional space. Since we cannot directly draw a four-dimensional graph, we usually study level surfaces:

\[ f(x,y,z)=c. \]

6.9.1 Worked example 6.12: Spherical level surfaces

Let

\[ f(x,y,z)=x^2+y^2+z^2. \]

The level surface \(f(x,y,z)=c\) is

\[ x^2+y^2+z^2=c. \]

For \(c>0\), this is a sphere of radius \(\sqrt c\) centered at the origin.

import numpy as np
import matplotlib.pyplot as plt

u = np.linspace(0, 2*np.pi, 100)
v = np.linspace(0, np.pi, 50)
U, V = np.meshgrid(u, v)
radius = 2
X = radius*np.cos(U)*np.sin(V)
Y = radius*np.sin(U)*np.sin(V)
Z = radius*np.cos(V)

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, alpha=0.65, linewidth=0)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_zlabel('$z$')
ax.set_title('Spherical level surface')
plt.show()

6.9.2 Worked example 6.13: Plane level surfaces

Let

\[ f(x,y,z)=2x-y+3z. \]

The level surface \(f(x,y,z)=c\) is

\[ 2x-y+3z=c. \]

For each value of \(c\), this is a plane. Different values of \(c\) give parallel planes.

6.9.3 Worked example 6.14: Ellipsoid level surfaces

Let

\[ f(x,y,z)=\frac{x^2}{4}+\frac{y^2}{9}+z^2. \]

The level surface \(f(x,y,z)=1\) is

\[ \frac{x^2}{4}+\frac{y^2}{9}+z^2=1, \]

an ellipsoid with semi-axis lengths \(2\), \(3\), and \(1\).

6.10 6.8 Functions on high-dimensional spaces

In statistics and machine learning, a function often has many variables. Suppose a regression model predicts

\[ \widehat y_i=\beta_0+\beta_1x_i. \]

The least-squares loss is

\[ L(\beta_0,\beta_1)=\sum_{i=1}^m \left(y_i-(\beta_0+\beta_1x_i)\right)^2. \]

This is a function

\[ L:\mathbb R^2\to\mathbb R. \]

For a model with \(p\) predictors, the parameter vector is

\[ \beta=(\beta_0,\beta_1,\ldots,\beta_p), \]

and the loss becomes a function

\[ L:\mathbb R^{p+1}\to\mathbb R. \]

We cannot draw this graph when \(p\) is large, but we can still study slices, contours, gradients, and local quadratic approximations.

import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(6)
x_data = np.linspace(-2, 2, 40)
y_data = 1.5 + 2.0*x_data + rng.normal(scale=0.35, size=x_data.shape)

b0 = np.linspace(-1.0, 4.0, 180)
b1 = np.linspace(-0.5, 4.5, 180)
B0, B1 = np.meshgrid(b0, b1)
Loss = np.zeros_like(B0)
for x_i, y_i in zip(x_data, y_data):
    Loss += (y_i - (B0 + B1*x_i))**2

fig, ax = plt.subplots()
cs = ax.contour(B0, B1, Loss, levels=24)
ax.clabel(cs, inline=True, fontsize=8)
ax.set_xlabel(r'$\beta_0$')
ax.set_ylabel(r'$\beta_1$')
ax.set_title('Least-squares loss contours')
ax.grid(True, alpha=0.25)
plt.show()

6.10.1 Likelihood functions

In statistics, a likelihood function assigns to each parameter value a number measuring how compatible that parameter is with the observed data. For example, for a normal model with known variance, the log-likelihood has the form

\[ \ell(\mu)=-\frac{1}{2\sigma^2}\sum_{i=1}^m (x_i-\mu)^2+\text{constant}. \]

For a multivariate model, \(\theta\) may have many components, and the log-likelihood becomes

\[ \ell(\theta):\mathbb R^n\to\mathbb R. \]

Thus optimization in statistics is also multivariable calculus.

6.11 6.9 Representations of a multivariable function

A multivariable function can be represented in several complementary ways.

Representation Meaning Example
Verbal description in words temperature depends on longitude and latitude
Algebraic formula \(f(x,y)=x^2+9y^2\)
Numerical table or data grid measured temperatures on a map
Visual surface, contour, heatmap graph of \(z=f(x,y)\)
Algorithmic code that computes values simulation or ML model

Modern applied mathematics often moves among all five. Data are numerical, models are algebraic or algorithmic, and interpretation requires visual and verbal reasoning.

6.12 6.10 Common surfaces and level sets

Equation Shape Key feature
\(z=ax+by+c\) plane linear surface
\(z=x^2+y^2\) circular paraboloid opens upward
\(z=x^2+9y^2\) elliptic paraboloid different curvature in different directions

6.13 6.11 Worked synthesis: one function, many views

Consider

\[ f(x,y)=e^{-x^2-y^2}. \]

Domain. The exponential function is defined for every real number, so the domain is all of \(\mathbb R^2\).

Range. Since \(x^2+y^2\ge 0\), we have \(-x^2-y^2\le 0\), so

\[ 0<f(x,y)\le 1. \]

The maximum value is \(1\) at \((0,0)\).

Graph. The graph is a bell-shaped surface centered at the origin.

Level curves. A level curve satisfies

\[ e^{-x^2-y^2}=c. \]

For \(0<c\le 1\), taking natural logarithms gives

\[ x^2+y^2=-\ln c. \]

Thus the level curves are circles centered at the origin.

6.14 6.12 Common mistakes

WarningMistake 1: Confusing graph and domain

The domain of \(f(x,y)\) is a region in the \(xy\)-plane. The graph of \(f\) is a surface in \(xyz\)-space. For example, the domain of \(\sqrt{4-x^2-y^2}\) is a disk, while the graph is a hemisphere.

WarningMistake 2: Forgetting strict inequalities

For logarithms, the input must be positive, not merely nonnegative. Thus \(\ln(4-x^2-y^2)\) has domain \(x^2+y^2<4\), not \(x^2+y^2\le 4\).

WarningMistake 3: Thinking all level curves are circles

Level curves may be lines, ellipses, hyperbolas, logarithmic curves, or more complicated curves. Their shape depends on the function.

## 6.1 Functions of several variables

A **function of two variables** is a rule that assigns to each input pair $(x,y)$ in a domain $D\subseteq \mathbb R^2$ exactly one real number $f(x,y)$:

$$
f:D\subseteq \mathbb R^2\to \mathbb R.
$$

The set $D$ is the **domain**. The set of all output values is the **range**:

$$
\operatorname{Range}(f)=\{f(x,y):(x,y)\in D\}.
$$

A **function of three variables** is a rule

$$
f:E\subseteq \mathbb R^3\to \mathbb R,
\qquad (x,y,z)\mapsto f(x,y,z).
$$

More generally, a **function of $n$ variables** is a rule

$$
f:D\subseteq \mathbb R^n\to \mathbb R,
\qquad x=(x_1,x_2,\ldots,x_n)\mapsto f(x_1,x_2,\ldots,x_n).
$$

A function $f:\mathbb R^n\to\mathbb R$ is often called a **scalar field** because it assigns a scalar value to each point in space.

### Examples from science, statistics, and AI

| Function | Input | Output | Interpretation |
|---|---:|---:|---|
| $T(x,y)$ | longitude and latitude | temperature | weather map |
| $p(x,y,z)$ | point in space | pressure | fluid or atmosphere |
| $\rho(x,y,z)$ | point in space | density | mass distribution |
| $L(\beta_0,\beta_1)$ | model parameters | squared error loss | linear regression |
| $\ell(\theta)$ | statistical parameter | log-likelihood | inference |
| $J(w_1,\ldots,w_n)$ | model weights | training objective | machine learning |

::: {.callout-tip title='Modern viewpoint'}
In data science and AI, the most important functions are often high-dimensional. A neural-network loss function may depend on millions or billions of parameters. We cannot draw its graph, but the ideas of domains, level sets, slices, gradients, and local approximations remain central.
:::

## 6.2 Domains as regions

Finding the domain is often the first step in studying a multivariable function. The algebraic restrictions are familiar from single-variable calculus, but now they describe regions in the plane or space.

Common restrictions:

| Expression | Restriction |
|---|---|
| $\sqrt{g(x,y)}$ | $g(x,y)\ge 0$ |
| $\ln(g(x,y))$ | $g(x,y)>0$ |
| $\dfrac{1}{g(x,y)}$ | $g(x,y)\ne 0$ |
| $\arcsin(g(x,y))$ | $-1\le g(x,y)\le 1$ |

### Worked example 6.1: Square root and denominator

Find the domain of

$$
f(x,y)=\frac{\sqrt{y-x}}{y}.
$$

The square root requires

$$
y-x\ge 0,
$$

so $y\ge x$. The denominator requires $y\ne 0$. Therefore

$$
D=\{(x,y)\in\mathbb R^2:y\ge x,\; y\ne 0\}.
$$

This is the half-plane above the line $y=x$, with the portion of the line $y=0$ removed.

### Worked example 6.2: Parabolic domain with excluded lines

Find the domain of

$$
g(x,y)=\frac{\sqrt{x^2-y}}{x^2-4}.
$$

The square root requires

$$
x^2-y\ge 0,
$$

so $y\le x^2$. The denominator requires

$$
x^2-4\ne 0,
$$

so $x\ne 2$ and $x\ne -2$. Thus

$$
D=\{(x,y)\in\mathbb R^2:y\le x^2,\; x\ne \pm 2\}.
$$

#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

## 6.3 Graphs of functions of two variables

For a function

$$
f:D\subseteq\mathbb R^2\to\mathbb R,
$$

the **graph** is the set of points in $\mathbb R^3$ given by

$$
\{(x,y,z):(x,y)\in D,\; z=f(x,y)\}.
$$

We usually write the graph as

$$
z=f(x,y).
$$

The graph is a surface above or below the domain in the $xy$-plane.

### Worked example 6.4: A plane as a graph

Let

$$
f(x,y)=6-2x-3y.
$$

Then

$$
z=6-2x-3y,
$$

or equivalently

$$
2x+3y+z=6.
$$

This is a plane. It has $x$-intercept $3$, $y$-intercept $2$, and $z$-intercept $6$.

Let

4-x^2-y^2\ge 0,
$$

so

$$
x^2+y^2\le 4.

$$
z=\sqrt{4-x^2-y^2}.

Squaring both sides gives

x^2+y^2+z^2=4.

## 6.4 Slices and traces

A surface can be understood by taking lower-dimensional slices.

For $z=f(x,y)$:

- fixing $y=y_0$ gives the curve $z=f(x,y_0)$ in a vertical plane parallel to the $xz$-plane;
- fixing $x=x_0$ gives the curve $z=f(x_0,y)$ in a vertical plane parallel to the $yz$-plane;
- fixing $z=c$ gives the level curve $f(x,y)=c$ in the $xy$-plane.

Slices connect single-variable calculus to multivariable calculus. Partial derivatives in later chapters measure slopes of coordinate slices.

### Worked example 6.8: Slices of an elliptic paraboloid

For

$$
z=x^2+9y^2,
$$

fixing $y=0$ gives

$$
z=x^2,
$$

while fixing $x=0$ gives

$$
z=9y^2.
$$

The second slice is steeper.

## 6.5 Level curves and contour maps

A **level curve** of a function $f(x,y)$ is a curve of the form

$$
f(x,y)=c,
$$

where $c$ is a constant in the range of $f$.

Geometrically, the level curve $f(x,y)=c$ is obtained by intersecting the surface $z=f(x,y)$ with the horizontal plane $z=c$ and projecting the intersection down to the $xy$-plane.

Contour maps are collections of level curves. They are used in topography, weather maps, probability density plots, and loss landscapes.

::: {.callout-tip title='How to read contours'}
Where level curves are close together, the function changes rapidly. Where level curves are far apart, the function changes slowly.
:::

### Worked example 6.9: Circular level curves

Let

$$
f(x,y)=x^2+y^2.
$$

The level curve $f(x,y)=c$ is

$$
x^2+y^2=c.
$$

For $c>0$, this is a circle centered at the origin with radius $\sqrt c$.

### Worked example 6.10: Elliptic level curves

Let

$$
f(x,y)=x^2+9y^2.
$$

The level curve $f(x,y)=c$ is

$$
x^2+9y^2=c.
$$

For $c>0$, these are ellipses.

### Worked example 6.11: Exponential level curves

Let

$$
f(x,y)=xe^{4y}.
$$

A level curve $f(x,y)=c$ satisfies

$$
xe^{4y}=c.
$$

If $x\ne 0$ and $c/x>0$, then

$$
y=\frac14\ln\left(\frac{c}{x}\right).
$$

This shows that level curves need not be lines, circles, parabolas, or ellipses.

```{python}
#| label: fig-ch06-exponential-levels

ax.clabel(cs, inline=True, fontsize=9)
ax.axvline(0, linewidth=1)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('Exponential level curves')
ax.grid(True, alpha=0.25)
plt.show()
```

## 6.6 Heatmaps and data surfaces

A multivariable function may be known through data rather than a formula. For example, a temperature field may be measured on a grid of locations, or a loss function may be evaluated at many parameter values.

A **heatmap** colors the plane according to function values. A **contour plot** draws level curves. Both are useful when a full surface plot is difficult to read.

```{python}
#| label: fig-ch06-heatmap-contour
#| fig-cap: 'A heatmap and contours for a smooth scalar field.'
#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 300)
y = np.linspace(-3, 3, 300)
X, Y = np.meshgrid(x, y)
Z = np.exp(-0.4*((X-1)**2 + (Y+0.5)**2)) + 0.7*np.exp(-0.8*((X+1.2)**2 + (Y-1.1)**2))

fig, ax = plt.subplots()
image = ax.imshow(Z, extent=[x.min(), x.max(), y.min(), y.max()], origin='lower', aspect='auto')
cs = ax.contour(X, Y, Z, levels=8, linewidths=1)
ax.clabel(cs, inline=True, fontsize=8)
fig.colorbar(image, ax=ax, label='function value')
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('Heatmap with contour lines')
plt.show()
```

## 6.7 Functions of three variables and level surfaces

A function of three variables has the form

$$
f(x,y,z).
$$

Its graph would be

$$
w=f(x,y,z),
$$

which lives in four-dimensional space. Since we cannot directly draw a four-dimensional graph, we usually study **level surfaces**:

$$
f(x,y,z)=c.
$$

### Worked example 6.12: Spherical level surfaces

Let

$$
f(x,y,z)=x^2+y^2+z^2.
$$

The level surface $f(x,y,z)=c$ is

$$
x^2+y^2+z^2=c.
$$

For $c>0$, this is a sphere of radius $\sqrt c$ centered at the origin.

```{python}
#| label: fig-ch06-sphere-level-surface
#| fig-cap: 'A level surface of $f(x,y,z)=x^2+y^2+z^2$ is a sphere.'
#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

u = np.linspace(0, 2*np.pi, 100)
v = np.linspace(0, np.pi, 50)
U, V = np.meshgrid(u, v)
radius = 2
X = radius*np.cos(U)*np.sin(V)
Y = radius*np.sin(U)*np.sin(V)
Z = radius*np.cos(V)

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, alpha=0.65, linewidth=0)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_zlabel('$z$')
ax.set_title('Spherical level surface')
plt.show()
```

### Worked example 6.13: Plane level surfaces

Let

$$
f(x,y,z)=2x-y+3z.
$$

The level surface $f(x,y,z)=c$ is

$$
2x-y+3z=c.
$$

For each value of $c$, this is a plane. Different values of $c$ give parallel planes.

### Worked example 6.14: Ellipsoid level surfaces

Let

$$
f(x,y,z)=\frac{x^2}{4}+\frac{y^2}{9}+z^2.
$$

The level surface $f(x,y,z)=1$ is

$$
\frac{x^2}{4}+\frac{y^2}{9}+z^2=1,
$$

an ellipsoid with semi-axis lengths $2$, $3$, and $1$.

## 6.8 Functions on high-dimensional spaces

In statistics and machine learning, a function often has many variables. Suppose a regression model predicts

$$
\widehat y_i=\beta_0+\beta_1x_i.
$$

The least-squares loss is

$$
L(\beta_0,\beta_1)=\sum_{i=1}^m \left(y_i-(\beta_0+\beta_1x_i)\right)^2.
$$

This is a function

$$
L:\mathbb R^2\to\mathbb R.
$$

For a model with $p$ predictors, the parameter vector is

$$
\beta=(\beta_0,\beta_1,\ldots,\beta_p),
$$

and the loss becomes a function

$$
L:\mathbb R^{p+1}\to\mathbb R.
$$

We cannot draw this graph when $p$ is large, but we can still study slices, contours, gradients, and local quadratic approximations.

```{python}
#| label: fig-ch06-loss-landscape
#| fig-cap: 'A two-parameter least-squares loss landscape. The best-fit parameters lie near the smallest contour.'
#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(6)
x_data = np.linspace(-2, 2, 40)
y_data = 1.5 + 2.0*x_data + rng.normal(scale=0.35, size=x_data.shape)

b0 = np.linspace(-1.0, 4.0, 180)
b1 = np.linspace(-0.5, 4.5, 180)
B0, B1 = np.meshgrid(b0, b1)
Loss = np.zeros_like(B0)
for x_i, y_i in zip(x_data, y_data):
    Loss += (y_i - (B0 + B1*x_i))**2

fig, ax = plt.subplots()
cs = ax.contour(B0, B1, Loss, levels=24)
ax.clabel(cs, inline=True, fontsize=8)
ax.set_xlabel(r'$\beta_0$')
ax.set_ylabel(r'$\beta_1$')
ax.set_title('Least-squares loss contours')
ax.grid(True, alpha=0.25)
plt.show()
```

### Likelihood functions

In statistics, a likelihood function assigns to each parameter value a number measuring how compatible that parameter is with the observed data. For example, for a normal model with known variance, the log-likelihood has the form

$$
\ell(\mu)=-\frac{1}{2\sigma^2}\sum_{i=1}^m (x_i-\mu)^2+\text{constant}.
$$

For a multivariate model, $\theta$ may have many components, and the log-likelihood becomes

$$
\ell(\theta):\mathbb R^n\to\mathbb R.
$$

Thus optimization in statistics is also multivariable calculus.

## 6.9 Representations of a multivariable function

A multivariable function can be represented in several complementary ways.

| Representation | Meaning | Example |
|---|---|---|
| Verbal | description in words | temperature depends on longitude and latitude |
| Algebraic | formula | $f(x,y)=x^2+9y^2$ |
| Numerical | table or data grid | measured temperatures on a map |
| Visual | surface, contour, heatmap | graph of $z=f(x,y)$ |
| Algorithmic | code that computes values | simulation or ML model |

Modern applied mathematics often moves among all five. Data are numerical, models are algebraic or algorithmic, and interpretation requires visual and verbal reasoning.

## 6.10 Common surfaces and level sets

| Equation | Shape | Key feature |
|---|---|---|
| $z=ax+by+c$ | plane | linear surface |
| $z=x^2+y^2$ | circular paraboloid | opens upward |
| $z=x^2+9y^2$ | elliptic paraboloid | different curvature in different directions |
| $z=x^2-y^2$ | saddle | rises in one direction, falls in another |
| $x^2+y^2+z^2=r^2$ | sphere | level surface of distance squared |
| $x^2+y^2=r^2$ | cylinder | no restriction on $z$ |
| $z=\sqrt{r^2-x^2-y^2}$ | upper hemisphere | graph over a disk |
| $x^2/a^2+y^2/b^2+z^2/c^2=1$ | ellipsoid | stretched sphere |

Recognizing these shapes will help with optimization, multiple integration, and vector calculus.

## 6.11 Worked synthesis: one function, many views

Consider

$$
f(x,y)=e^{-x^2-y^2}.
$$

**Domain.** The exponential function is defined for every real number, so the domain is all of $\mathbb R^2$.

**Range.** Since $x^2+y^2\ge 0$, we have $-x^2-y^2\le 0$, so

$$
0<f(x,y)\le 1.
$$

The maximum value is $1$ at $(0,0)$.

**Graph.** The graph is a bell-shaped surface centered at the origin.

**Level curves.** A level curve satisfies

$$
e^{-x^2-y^2}=c.
$$

For $0<c\le 1$, taking natural logarithms gives

$$
x^2+y^2=-\ln c.
$$

Thus the level curves are circles centered at the origin.

```{python}
#| label: fig-ch06-gaussian-levels
#| fig-cap: 'The surface $e^{-x^2-y^2}$ and its circular level curves.'
#| fig-width: 6.5
#| fig-height: 5
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 300)
y = np.linspace(-3, 3, 300)
X, Y = np.meshgrid(x, y)
Z = np.exp(-X**2 - Y**2)

fig, ax = plt.subplots()
image = ax.imshow(Z, extent=[-3, 3, -3, 3], origin='lower', aspect='equal')
cs = ax.contour(X, Y, Z, levels=[0.05, 0.1, 0.2, 0.4, 0.7, 0.9])
ax.clabel(cs, inline=True, fontsize=8)
fig.colorbar(image, ax=ax, label='$f(x,y)$')
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_title('Gaussian-shaped scalar field')
plt.show()
```

This example is a first glimpse of the geometry behind multivariate normal densities, radial basis functions, kernel methods, and smooth energy landscapes.

## 6.12 Common mistakes

::: {.callout-warning title='Mistake 1: Confusing graph and domain'}
The domain of $f(x,y)$ is a region in the $xy$-plane. The graph of $f$ is a surface in $xyz$-space. For example, the domain of $\sqrt{4-x^2-y^2}$ is a disk, while the graph is a hemisphere.
:::

::: {.callout-warning title='Mistake 2: Forgetting strict inequalities'}
For logarithms, the input must be positive, not merely nonnegative. Thus $\ln(4-x^2-y^2)$ has domain $x^2+y^2<4$, not $x^2+y^2\le 4$.
:::

::: {.callout-warning title='Mistake 3: Thinking all level curves are circles'}
Level curves may be lines, ellipses, hyperbolas, logarithmic curves, or more complicated curves. Their shape depends on the function.
:::

## 6.15 Chapter summary

A function of several variables assigns one scalar output to each input point in a multidimensional domain:

$$
f:D\subseteq\mathbb R^n\to\mathbb R.
$$

The main ideas of this chapter are:

- domains of multivariable functions are regions in $\mathbb R^n$;
- the graph of $z=f(x,y)$ is a surface in $\mathbb R^3$;
- level curves $f(x,y)=c$ show where a two-variable function is constant;
- level surfaces $f(x,y,z)=c$ play the same role for functions of three variables;
- slices and traces reduce multivariable functions to single-variable views;
- high-dimensional functions in statistics and AI are studied through contours, slices, heatmaps, gradients, and optimization paths.
